# Changelog

## 0.3.0

### Added

- support for specifiyng how options should be formatted(snake case or camel case)

## 0.2.0

### Removed

- `WithSettings` method- now it is done automatically for classes that are parameters in the added functions.

### Changed

- Automatically generate options from classes that are parameters in the functions added to the `CommandBuilder`.
- Options are added to the commands that are going to use them, not every command.

### Added

- Extra description in the `README.md` file.
- More types are turned automatically into Options and Arguments.
